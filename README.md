## node.js and express for backend, react with vite as frontend framework & mongodb for database

# how to run
- clone the repo
- cd client
- npm i
- npm run dev
- front end is hosted at localhost:5173
- open new terminal
- cd server 
- npm i
- npm run dev
- backend is hosted at localhost:5000
